📋 项目简介

OfficeCLI 是一个开源、单二进制文件的命令行工具，无需安装 Office 即可在任意平台创建、读取和修改 Word、Excel 和 PowerPoint 文档。

本版本为安全加固版，针对企业商用场景进行了全面的安全审计和漏洞修复，确保符合 OWASP Top 10 安全标准。

🛡️
命令注入防护
所有用户输入经过严格验证，防止恶意命令执行
🔐
路径遍历检测
检测并阻止 ..、~ 等路径遍历攻击模式
🧹
安全临时文件
使用 GUID 命名临时文件，防止符号链接攻击
🌐
XSS 攻击防护
HTML 输出经过严格转义和过滤
📡
网络访问控制
仅绑定本地连接，防止远程攻击
🔒
禁用自动更新
防止供应链攻击，确保版本稳定性

🔒 安全修复详情

✅
临时文件路径可预测性漏洞

使用 GUID 替代时间戳+文件名，防止符号链接攻击

✅
MCP 服务器路径遍历漏洞

添加 SafeArg() 验证，检测危险字符和路径模式

✅
进程执行安全

使用安全的 ProcessStartInfo 配置

✅
HTML 输出 XSS 风险

添加 HtmlSanitizer.cs 组件和 CSP 安全策略

✅
自动更新功能禁用

完全禁用网络更新调用，防止远程代码执行

✅
文件权限安全

设置最小必要权限 (755)

🚀 快速开始

# macOS / Linux 安装

curl -fsSL https://raw.githubusercontent.com/13888285815/codebuddy-OfficeCLI/security-hardened-demo/install.sh | bash


# Windows (PowerShell)

irm https://raw.githubusercontent.com/13888285815/codebuddy-OfficeCLI/security-hardened-demo/install.ps1 | iex


# 创建演示文稿

officecli create deck.pptx

officecli add deck.pptx / --type slide --prop title="Hello, World!"
