# 隐私政策

## 概述

OfficeCLI 商用安全版本（以下简称"本软件"）高度重视用户隐私保护。本隐私政策说明了我们如何收集、使用和保护您的信息。

**核心原则**：本软件在设计上最大限度地减少数据收集，所有处理均在本地完成。

## 我们不收集的数据

本软件**不会**收集以下任何信息：

- ❌ 文档内容
- ❌ 文档元数据
- ❌ 使用统计或遥测数据
- ❌ 崩溃报告
- ❌ 网络活动日志
- ❌ 系统信息
- ❌ 用户身份信息

## 本地存储的数据

本软件仅在您的本地设备上存储以下信息：

### 1. 配置文件 (`~/.officecli/config.json`)

**存储内容**：
- 日志启用状态（true/false）
- 已安装二进制版本号

**不存储**：
- 文档历史
- 使用记录
- 个人身份信息

**文件权限**：
- Unix/Linux/macOS：仅限当前用户访问（权限 600）
- Windows：继承用户配置文件权限

### 2. 日志文件 (`~/.officecli/officecli.log`)

**仅在启用日志功能时创建**

**存储内容**：
- 执行的命令（已脱敏处理）
- 时间戳

**脱敏处理**：
- 文件路径仅保留文件名
- 敏感关键词被替换为 `[REDACTED]`
- 输出内容长度限制在 1000 字符以内

**默认状态**：禁用

### 3. 临时文件

**位置**：系统临时目录

**用途**：
- HTML 预览文件
- 文档处理中间文件

**生命周期**：
- 大多数临时文件在使用后立即删除
- 建议定期清理系统临时目录

## 网络活动

### 本软件不会：

- 连接自动更新服务器
- 发送遥测数据
- 上传文档到云端
- 下载外部代码（除用户明确发起的图片下载）

### 用户发起的网络请求：

以下功能可能涉及网络请求，但仅在用户明确操作时发生：

| 功能 | 说明 | 数据发送 |
|------|------|---------|
| 插入网络图片 | 下载用户指定的图片 URL | 仅 HTTP 请求头 |

## 第三方服务

本软件**不使用**任何第三方服务：
- 无分析服务（Google Analytics 等）
- 无崩溃报告服务（Sentry 等）
- 无云服务（AWS、Azure 等）
- 无 CDN

## 数据安全

### 保护措施

1. **本地处理**：所有文档处理在本地完成
2. **无持久化**：不保存文档内容到磁盘（除用户保存操作）
3. **内存安全**：使用安全的内存管理实践
4. **权限控制**：配置文件设置严格的文件权限

### 建议的安全实践

作为用户，您可以采取以下措施增强安全性：

```bash
# 1. 定期检查配置文件权限
ls -la ~/.officecli/

# 2. 定期清理日志（如果启用）
officecli config log clear

# 3. 清理临时文件
# macOS
rm -rf $TMPDIR/officecli_*

# Linux
rm -rf /tmp/officecli_*

# Windows
Remove-Item $env:TEMP\officecli_* -Recurse
```

## 数据保留

- **配置文件**：永久保留，直到用户删除
- **日志文件**：最大 1MB，自动轮转
- **临时文件**：立即删除或随系统清理

## 用户权利

您拥有以下权利：

1. **知情权**：本隐私政策详细说明了数据处理
2. **访问权**：可随时查看配置文件内容
3. **删除权**：可随时删除配置文件和日志
4. **选择权**：可选择不启用日志功能

## 儿童隐私

本软件不面向 13 岁以下儿童，我们不会故意收集儿童的个人信息。

## 政策更新

我们可能会更新本隐私政策。更新将：
- 发布在 GitHub 仓库
- 标注更新日期
- 重大变更将在 Release Notes 中说明

## 合规性

本软件设计符合以下隐私原则：
- GDPR（欧盟通用数据保护条例）
- CCPA（加州消费者隐私法案）
- 数据最小化原则

## 联系与报告

### 隐私问题

如有隐私相关问题，请通过 GitHub Issues 联系。

### 安全漏洞报告

如果您发现安全问题：
1. **不要**公开披露
2. 私下联系维护者
3. 提供详细的复现步骤

## 总结

OfficeCLI 商用安全版本的设计理念是**最小化数据收集**：

- ✅ 所有处理本地完成
- ✅ 不收集个人数据
- ✅ 不连接外部服务
- ✅ 用户完全控制

我们相信隐私是基本权利，本软件的设计体现了这一承诺。
